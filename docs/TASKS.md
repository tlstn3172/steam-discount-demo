### 1.4 핵심 비즈니스 로직 구현 (Week 2-3)

#### 1.4.1 VPI (가성비 지수) 계산 로직
- [ ] VPI 계산 함수 구현
  ```python
  def calculate_vpi(playtime: float, price: float) -> float:
      """VPI = (예상 플레이 시간 / 현재 가격) × 1000"""
      if price == 0:
          return 0
      return (playtime / price) * 1000
  ```
- [ ] 예외 처리 (가격이 0인 경우, 플레이 시간 데이터 없는 경우)
- [ ] VPI 순위 계산 로직
- [ ] 단위 테스트 작성

#### 1.4.2 스마트 구매 신호등 로직
- [ ] 구매 신호 계산 함수 구현
  ```python
  def calculate_purchase_signal(current_price: float, all_time_low: float) -> str:
      """초록/노랑/빨강 신호 반환"""
      if current_price <= all_time_low:
          return "BEST"
      elif (current_price - all_time_low) / all_time_low <= 0.1:
          return "GOOD"
      else:
          return "WAIT"
  ```
- [ ] 신호별 메시지 매핑
- [ ] 단위 테스트 작성
